Create a complete production-ready Cricket Tournament Management Platform using Node.js for the backend and React.js (Next.js preferred) for the frontend.

## Project Name

Cricket Tournament Hub

## Objective

Build a professional cricket tournament website where teams can register for tournaments, add all 11 players, manage player statistics, and allow visitors to view player and team histories.

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB with Mongoose
* JWT Authentication
* Multer for image uploads
* Cloudinary/AWS S3 for storing profile pictures
* REST API architecture
* Swagger API documentation

### Frontend

* Next.js (latest version)
* Tailwind CSS
* Axios
* React Query
* Redux Toolkit (if needed)
* Responsive Design (Mobile + Desktop)

### Admin Panel

* Tournament Management
* Team Approval/Rejection
* Player Verification
* Match Scheduling
* Score Entry
* Statistics Management
* Dashboard Analytics

For each player:

* Date of Birth
* Batting Style
* Bowling Style
* Role

  * Batsman
  * Bowler
  * All-Rounder
  * Wicket Keeper
* Jersey Number
* Profile Picture

Validation:

* Exactly 11 players required
* Captain must be one of the players

After registration:
